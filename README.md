# Musicstory

A website for exploring music stories and discovering the fascinating stories behind songs.

## Features:
- TikTok-style comment system with @mentions
- Song like buttons with dynamic ranking
- Top Hits page ranked by community engagement
- Search functionality
- Responsive design

## Files included:
- index.html (homepage)
- explore.html (explore page with comment system)
- hits.html (top hits ranked by likes)
- homepage.css
- explore.css
- hits.css
- pics/ folder with images
- vercel.json for deployment configuration

## Deployment:
Make sure all files including the pics/ folder are included in your deployment.
